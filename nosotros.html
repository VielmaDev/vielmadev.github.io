{ AstrosView }

AstrosView es un website desarrollado bajo la modalidad de práctica profesional, que utiliza una API llamada APOD (Astronomy Picture of the Day), perteneciente a la agencia espacial NASA. El website a través de esta API, permite conocer y explorar en primer plano las imágenes astronómicas que son publicadas diariamente por la agencia espacial. El contenido a mostrar es un artículo de prensa de carácter científico con su respectiva imagen de autoría. Cuenta con un motor de búsqueda tipo calendario, que permite filtrar en la galería de la API, la imagen de acuerdo a la fecha en que fue publicada. Si deseas conocer más sobre este proyecto ingresa a el perfil de Github AstrosView
